Leçon : La double distributivité

1. Définition

La double distributivité est une propriété qui permet de développer le produit de deux sommes :

Formule générale :
(a + b)(c + d) = a·c + a·d + b·c + b·d

2. Exemples

Exemple 1 :

Développons (x + 2)(x + 3) :

  1. x·x = x2
  2. x·3 = 3x
  3. 2·x = 2x
  4. 2·3 = 6

Résultat : x2 + 3x + 2x + 6 = x2 + 5x + 6

Exemple 2 :

Développons (2a – b)(3a + 4b) :

  1. 2a·3a = 6a2
  2. 2a·4b = 8ab
  3. (–b)·3a = –3ab
  4. (–b)·4b = –4b2

Résultat : 6a2 + 8ab – 3ab – 4b2 = 6a2 + 5ab – 4b2

3. Exercices

1. Développer (m + 5)(m + 2).

2. Développer (3x – 4)(2x + 1).

Voir les solutions

4. Solutions

Solution 1 :

(m + 5)(m + 2) = m·m + m·2 + 5·m + 5·2 = m2 + 2m + 5m + 10 = m2 + 7m + 10

Solution 2 :

(3x – 4)(2x + 1) = 3x·2x + 3x·1 – 4·2x – 4·1 = 6x2 + 3x – 8x – 4 = 6x2 – 5x – 4
